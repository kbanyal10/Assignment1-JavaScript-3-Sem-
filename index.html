Instructions to use the game

  1. Click buttons to see different actions.
  2. The purple button selects any button from list of first column.
  3. The sky blue button selects any button from list of second column.
  4. The green button selects any button from list of third column.
  5. The orange button selects any button from list of fourth column.
  6. The pink button selects any button from list of five column.
  7. The red button selects the reset function and resets the String.
  8. The sky blue button, adjacent to red button which will surprizingly say a sentence.
  9. To view whats going on background please see the console.
